# JOJO Minesweeper - Golden Wind Edition

這是一個結合《JOJO的奇妙冒險：黃金之風》風格的現代化踩地雷遊戲。
不僅保留了經典的邏輯運算，還加入了替身攻擊特效與「時間刪除」悔棋功能。

## 🎮 遊戲特色

1.  **JOJO 風格視覺設計**：
    - 使用喬魯諾·喬巴拿的經典配色（黃金/深紫）。
    - 菱格紋背景與高級玻璃擬態 UI。
    - 義大利風格字體設計。

2.  **替身能力特效**：
    - **Game Over**：觸發「緋紅之王 (King Crimson)」的穿胸一擊。
    - **悔棋功能 (Undo)**：觸發「刪除時間」特效，讓時光倒流回安全狀態。

3.  **完整遊戲功能**：
    - 3 種難度選擇 (Easy/Medium/Hard)。
    - 首點安全保證 (First Click Safe)。
    - 右鍵插旗與計數器。
    - 計時器功能。

## 🚀 如何執行

1.  直接開啟 `index.html` 即可遊玩。
2.  若要部署，建議使用 **Vercel**：
    - 將專案上傳至 GitHub。
    - 在 Vercel 匯入 Repository。
    - 點擊 Deploy 即可獲得線上網址。

## 🕹️ 操作說明

- **左鍵點擊**：揭開格子。
- **右鍵點擊**：標記地雷 (🚩)。
- **踩到地雷**：
    - 觸發 Game Over 動畫。
    - 點擊角落出現的 **「KING CRIMSON!」** 按鈕即可發動能力回到上一步。

## 🛠️ 技術對棧

- HTML5
- CSS3 (Variables, Grid, Keyframe Animations)
- JavaScript (ES6+)
